# 📚 Library Management System

## 📌 Project Description

The Library Management System is a console-based Java application developed during the Pinnacle Labs Java Development Internship. It helps manage library books by allowing users to add, view, issue, and return books through an interactive menu-driven interface.

## ✨ Features

- Add New Books
- View All Books
- Issue Books
- Return Books
- Book Availability Status
- Menu-Driven Interface
- Input Validation
- Library Full Check

## 🛠️ Technologies Used

- Java
- VS Code
- Object-Oriented Programming (OOP)
- Arrays
- Loops
- Conditional Statements
- Scanner Class

## ▶️ How to Run

1. Open the project in VS Code.
2. Compile the Java files.
3. Run `LibraryManagementSystem.java`.
4. Select an option from the menu.
5. Manage books by adding, viewing, issuing, or returning them.

## 📸 Output

- Library Menu
- Add Book
- View Books
- Issue Book
- Return Book
- Exit Message
